Static site for a developer.

This project was my first ever created website.
Idea and starter package provided by Skillcrush's Front-End Developer course as a final project.

I really enjoy the light and clean basic design of this site. It gives the eye a break from all the clutter that could wound up on a site if we do not pay attention to details.

It was a great project to understand how the building of a simple site works. Everyone's gotta start somewhere and this right here was where I did. There is nothing to compare to building a website, especially if it is your first, even though it is static - it summarizes the base of Front-End.
