Projects made by me...

3D effect parallax site.

This site was an independent project for fun and to understand the making of parallax better.

I have wanted to create a parallax website and although it is not the traditional type - that's what makes it special, I really enjoyed the making of the 3D effect which serves as the parallax part.

The site itself is small as the main focus was on the making of the crowd and fireworks which was achieved with layers and positioning. I did small customizations, for example one of them is the sticky footer.
I would strongly advise to checkout the GitHub page created because the images below can not give back the scrolling 3D effect, not even remotely.

You can check the page live or checkout my repository on GitHub.
